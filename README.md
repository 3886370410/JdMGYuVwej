# Python数据分析可视化系统

## 核心功能
- 登录注册
- 数据总览
- 类型分析
- 价格分析
- 评价分析
- 地区分析
- 店铺名称词云图
- 评价预测

## 适用场景
- 美食行业数据分析
- 消费趋势研究
- 店铺评估与比较
- 市场营销策略制定

## 技术特点
- 使用Flask框架构建Web应用
- 数据库采用MySQL存储
- 前端界面设计基于HTML、CSS和JavaScript
- 提供数据可视化功能，直观展示分析结果
